# Análise de Vendas - Contoso (End-to-End)

## Objetivo

Analisar os fatores que impactam o desempenho de vendas, identificando oportunidades para aumento de receita e lucratividade.

## Tecnologias

* SQL Server
* Python (Pandas, Matplotlib, Seaborn)
* Power BI

## Fonte de Dados

Base **ContosoRetailDW**

## Pipeline do Projeto

SQL Server → Python → Power BI

## Análises Planejadas

* Evolução de vendas ao longo do tempo
* Produtos mais vendidos e mais lucrativos
* Performance por loja
* Análise por região
